# Module_bl0940_ESP32_SPI
## The esp32 library communicates with the bl0940 module using spi
  - SPI level is set to 900000 Hz (900 kHz)
## Connect pin
|     ESP32   |     BL0940   |
|:-----------:|:------------:|
|      23     |      SDI     |
|      19     |      SDO     |
|      18     |      SCLK    |

Note: Pin SEL chip BL0940 connect 3.3V
